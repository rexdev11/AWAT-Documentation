OfGeist

An Omni-Sentiment Political Prediction Toolset for traversing political and economic state in the United States of America. This framework holds the capacity to be built by asynchronous teams continually to improve output's quality“OfGeist”. This module will provide the exploration, discovery and building patterns used to standardize contribution and create the ability to asynchronously contribute. Discovering the scaffolding of this project will enable others to contribute to other parts the future project modules.

Tax Projecting

A tax projection is a point of reference calculative values based off the metrics and events over a time-line for both entities and tax procedures. This tool provides state for past present and future values for the entity.

AI Updater

The tax process is maintained and kept current through document discovery. This module will keep a watch on tax rules and updated the system with new or changed process as they are released to the public.
